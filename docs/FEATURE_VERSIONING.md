# Feature Versioning

Strict schema and software versioning are tracked to support replay, reproducibility, and explainability.

## Tracked Software Layers
- **Feature Pipeline Version**: Version of extraction code.
- **Registry Version**: Governance config version.
- **Validation Version**: Rules definition version.
- **Quality Version**: Quality math version.
- **Normalization Version**: Normalization reference bounds version.
- **Physics Engine Version**: Upstream characterization pipeline version.
