# ML Readiness

ML Readiness is evaluated at runtime to determine if the ingestion pipeline features are safe for model consumption.

## Readiness Checklist
1. **Pipeline Status**: API gateway services are healthy and stable.
2. **Feature Coverage**: 100% of registered features are extracted.
3. **Validation Rate**: At least 95% of features pass allowed ranges.
4. **Lineage Securement**: Feature vectors maintain strict traceability.
5. **Class Balance**: Warning checks for high imbalances are active.
